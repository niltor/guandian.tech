现在的问题是选择哪个DBaaS品牌。 SlashData调查了1,800多名后端开发人员，以确定市场上排名前15位的DBaaS品牌。报告参考了类似的可扩展性，易用性和开发，定价，支持/文档的速度，与其他系统集成，学习曲线，服务管理API的质量，以多帐户属性，根据该报告的小册子 。

以下是15个最受欢迎的DBaaS品牌。

  1. Microsoft Azure SQL数据库
  2. 亚马逊RDS
  3. 亚马逊DynamoDB
  4. Microsoft Azure表存储
  5. Microsoft DocumentDB
  6. MongoDB地图集
  7. Google Cloud SQL
  8. Google Cloud数据存储区
  9. 亚马逊极光
  10. Oracle数据库云服务
  11. Amazon SimpleDB
  12. Google Cloud Bigtable
  13. IBM Cloudant
  14. 撰写（IBM）
  15. Google Cloud Spanner

DBaaS简化了业务运营，并且通常无需高级内部IT人员来处理系统的复杂性。随着公司开始放弃内部服务器机房并依靠自动化来运行其数据库，DBaaS处于运营的最前沿。
